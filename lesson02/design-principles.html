Hick's Law

Google

google.com

Hick's Law states that the more choices a person is presented with, the longer the person will take to reach a decision. If users already know what they want to do before seeing the list of choices, the time it takes them to act is likely to be less. Therefore, a web designer should limit the options to the most important ones. In Google homepage, it simply provides a search bar in the webpage because its main function is to help users to search information on the Inthernet. It also provides a button so users can log in. The layout is neat and clean that follows Hick's Law.

PARC: Repetition

IKEA

ikea.com

Repetition means common elements, such as colors, graphics, layout styles, and so on, should be repeated to give a consistent look and feel to a page. IKEA uses bright colors for each card and bold text to describe each card with smaller text on top of the bold text as hints. All the text is positioned at the left-bottom corner.

PARC: Alignment

Khan Academy

khanacademy.org

Alignment means nothing should be placed on the page randomly. Every item on the page should be a visual connection with something else on the page. Khan Academy has a drop-down menu that shows a long list for users to choose the lesson they want to study. It's a well-organized webpage that follows PARC: alignment.
